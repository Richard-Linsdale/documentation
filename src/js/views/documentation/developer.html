The source code is available on GitHub.

The build/dependency manangement system is A Maven Stub, utilising the Oracle JET toolset and libraries (utilising many industry standard JavaScript tools).

The development environment uses the Netbeans IDE, but this is not mandatory, any development environment which can work with a Maven and OJET project.
